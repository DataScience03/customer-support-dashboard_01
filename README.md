# Customer Support Analytics Dashboard

## Overview

This project is a Streamlit-based dashboard for analyzing customer support tickets. It provides insights into ticket categories, priorities, and statuses through interactive visualizations.

## Features

* Upload support ticket data
* View ticket statistics
* Analyze ticket categories
* Track ticket priorities
* Interactive charts and visualizations

## Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib

## Installation

1. Clone the repository:

   ```
   git clone <repository-url>
   ```

2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

3. Run the application:

   ```
   streamlit run app.py
   ```
